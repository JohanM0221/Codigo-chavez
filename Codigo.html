    Cómo hacer un programa en C++ para diferenciar entre una cadena y un número

    En C++, podemos diferenciar entre una cadena y un número utilizando las funciones disponibles en la biblioteca estándar. En este ejemplo, vamos a crear un programa que le pedirá al usuario que ingrese un dato y determinará si es una cadena o un número.

    Paso 1: Incluir las bibliotecas necesarias

    Para trabajar con cadenas y números, necesitaremos incluir las bibliotecas string y iostream en nuestro programa:

                #include <string>
            #include <iostream>
        
            using namespace std;

    Paso 2: Crear una función para verificar si el dato es un número

    Podemos usar la función stoi() para convertir una cadena en un número entero. Si el dato ingresado no puede ser convertido en un número, la función lanzará una excepción. Para evitar esto, podemos crear una función que verifique si el dato es un número antes de intentar convertirlo:

                bool es_numero(const string& str) {
                for (char const & c : str) {
                    if (isdigit(c) == 0) return false;
                }
                return true;
            }

    Esta función recibe una cadena como parámetro y devuelve true si la cadena es un número, o false si no lo es.

    Paso 3: Crear el programa principal

    En el programa principal, utilizaremos un bucle for para pedir al usuario que ingrese un dato y verificar si es un número. Si el dato no es un número, se darán tres oportunidades para ingresar un valor válido antes de finalizar el programa:

                int main() {
                string dato;
                bool es_numero_flag = false;
                int intentos = 0;
        
                cout << "Ingrese un dato: ";
                cin >> dato;
        
                while (intentos < 3) {
                    if (es_numero(dato)) {
                        cout << "El dato ingresado es un número." << endl;
                        es_numero_flag = true;
                        break;
                    } else {
                        intentos++;
                        cout << "El dato ingresado no es un número. Intente nuevamente: ";
                        cin >> dato;
                    }
                }
        
                if (!es_numero_flag) {
                    cout << "Se han agotado los intentos. El programa se cerrará." << endl;
                }
        
                return 0;
            }

    En este programa, primero se pide al usuario que ingrese un dato. Luego, se llama a la función es_numero() para verificar si el dato es un número. Si es así, se muestra un mensaje indicando que el dato es un número y se establece la bandera es_numero_flag en true. Si no es un número, se le pide al usuario que ingrese el dato nuevamente y se incrementa la variable intentos. Si la variable intentos alcanza el valor de 3, se muestra un mensaje de error.
